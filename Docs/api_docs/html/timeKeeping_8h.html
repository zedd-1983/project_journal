void userTimeConfig ( void *  pvParameters)

FreeRTOS task for user defined time configuration.

this task stops disables the interrupt on UART0 first followed by stopping the RTC timer. User input is then obtained and verified and upon success loaded into rtc_datetime_t structure that holds values of current time and date thus changing it. Timer and UART0 interrupts are then enabled again. The task then deletes itself.
